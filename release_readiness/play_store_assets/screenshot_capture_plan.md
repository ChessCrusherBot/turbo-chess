# Turbo Chess Screenshot Capture Plan

Use screenshots from the latest installed release APK only.

Recommended phone screenshots:

1. Home screen showing Turbo Chess and main navigation.
2. Train screen showing Opening, Middlegame, Endgame, and Play vs Computer.
3. Opening Drills grid showing Jump to Position.
4. Active drill board.
5. Completed/checkmate dialog if available.
6. Computer Setup screen with sticky Start Game button.
7. Play vs Computer active board.
8. Game Replay screen with horizontal move strip.
9. More screen or About screen if an extra screenshot is needed.

Rules:

* No debug banner.
* No old ads/subscription/rewarded-pass text.
* No personal/private data.
* Use latest APK only.
* Portrait phone screenshots.
* Keep status bar clean if possible.
* Avoid showing legal walls as main marketing screenshots unless needed.
* Do not use Chess.com/Lichess/Stockfish logos.
* Do not edit screenshots in a misleading way.
