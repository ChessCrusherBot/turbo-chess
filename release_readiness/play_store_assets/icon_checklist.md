# Turbo Chess Play Store Icon Checklist

Required Play Store icon:

* 512 x 512 px
* PNG
* 32-bit PNG with alpha
* Maximum 1024 KB

Check:

* Icon matches Turbo Chess branding.
* Icon is not blurry.
* Icon is not copied from Chess.com, Lichess, Stockfish, or another brand.
* Icon does not include too much small text.
* Icon should look clear at small size.
* Keep source/export file in a safe branding folder.

Do not create a fake icon.
