# Turbo Chess Chess Sounds

Turbo Chess uses short CC0 click sounds for chess move feedback:

- `move.mp3`
- `capture.mp3`
- `check.mp3`
- `checkmate.mp3`

The sounds are intentionally short, non-musical, non-voice, and suitable for
repeated chess moves. See `SOURCE.md`, `LICENSES.md`, and `VERIFICATION.md`
before replacing or adding sound files.
