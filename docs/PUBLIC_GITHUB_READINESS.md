# Public GitHub Readiness

This document records a non-destructive public-release readiness pass for Turbo Chess. It does not contain secret values and is not legal advice.

## Current Readiness Status

Turbo Chess is close to public GitHub readiness from an engineering perspective. The repository has no commits yet, so there are currently no tracked files or tracked secret-like paths in Git history. A final manual review is still required before the first public commit and push.

## Files That Must Not Be Published

Do not publish:

- keystore files
- `android/key.properties`
- `android/keystore.properties`
- `android/local.properties`
- `.env` files
- service account JSON files
- `client_secret*.json`
- Play Console credentials
- bank/payment information
- private API keys or tokens
- generated `build/` output
- `.dart_tool/`
- Gradle caches
- local tool dumps such as `.qwen/`, `binding_lib.json`, `navigator_lib.json`, and `isolate_dump.json`
- generated screenshots unless deliberately reviewed for public documentation
- generated position-factory reports under `tools/position_factory/reports/`
- raw PGN/ZST database inputs

## Public GitHub Publish Steps

1. Review `git status`.
2. Review manual-review paths below.
3. Keep private or remove any secrets before staging.
4. Create a new public GitHub repository.
5. Add safe files only.
6. Commit with: `Prepare Turbo Chess GPLv3 open-source release`
7. Push.
8. Confirm GitHub shows `LICENSE`, `README.md`, and `THIRD_PARTY_NOTICES.md`.
9. Create a release tag matching the app version.
10. Continue Play Store release only after source availability is ready.

## Manual Review Before First Push

Review these before staging:

- `android/local.properties`: must stay private.
- `android/app/build.gradle.kts`: contains signing configuration structure and AdMob configuration plumbing; no password values should be committed.
- `ios/Runner/Info.plist`: contains AdMob identifier configuration; review before publishing if iOS source is included.
- `android/app/src/main/kotlin/com/turbochess/app/MainActivity.kt`: OAuth/raw URI logging was removed during manual-review cleanup. Re-scan before publishing if auth/deeplink handling is added later.
- `screenshots/`: generated screenshots; publish only if deliberately intended as public documentation.
- `tools/position_factory/reports/`: generated extraction reports; not required to build the app.
- `*.pgn`, `*.pgn.zst`, `*.zst`: raw database/input files; do not publish unless deliberately reviewed.
- `.qwen/`, `binding_lib.json`, `navigator_lib.json`, `isolate_dump.json`: local/tool-generated files; do not publish.

## Reminder

If a secret was ever committed to a public repository, rotate it. Removing it later is not enough.
