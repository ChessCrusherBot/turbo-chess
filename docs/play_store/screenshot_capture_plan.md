# Screenshot Capture Plan

Use screenshots from the current installed release build.

## Recommended phone screenshots

1. Home screen showing Turbo Chess and main navigation.
2. Train screen showing Opening, Middlegame, Endgame, and Play vs Computer.
3. Opening Drills grid showing Jump to Position.
4. Active drill board.
5. Completed/checkmate dialog if available.
6. Computer Setup screen with visible Start Game action.
7. Play vs Computer active board.
8. Game Replay screen with the move strip.
9. More screen or About screen if an extra screenshot is needed.

## Capture notes

Screenshots should use the current release UI, portrait phone framing, clean status bar state, and no personal/private data.

Marketing screenshots should not rely on legal text walls as the main visual. They should also avoid third-party logos, misleading edits, fake awards, and endorsement claims.
