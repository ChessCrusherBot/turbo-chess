# Store Listing Wording Notes

Store listing text and screenshots should match the current Android release behavior.

Recommended checks:

* App title fits store limits.
* Short description fits store limits.
* Full description matches app behavior.
* Screenshots come from the current release build.
* Screenshots do not show debug UI or personal/private data.
* Store copy avoids misleading ranking claims.
* Store copy avoids fake awards.
* Store copy avoids third-party brand comparison claims.
* Store copy does not claim Stockfish or Lichess endorsement.
* Store copy does not describe Turbo Chess as an official Stockfish or official Lichess app.
* Store copy matches the current free, ad-free, local-only app state.
