# Stockfish Source Availability Reminder

Turbo Chess includes Stockfish, licensed under GNU GPLv3.

The repository records:

* Stockfish upstream source URL
* Stockfish 18 release/tag/commit
* GPLv3 license text
* Android ABI paths
* Build notes
* Source correspondence notes

Maintainers should complete a Stockfish GPLv3/source availability check whenever distributing public app binaries, including Play Store builds.

This document does not certify legal compliance. It records the source-release maintenance item that should remain visible to maintainers.
