# Source License Summary

This document summarizes source/license information maintained for Turbo Chess v1.

## Summary

* Turbo Chess source code is released under GPLv3.
* GPLv3 license text is present in `LICENSE`, `COPYING`, `assets/legal/GPL-3.0.txt`, and `assets/stockfish/LICENSE.txt`.
* Stockfish source metadata and build notes are present.
* Cburnett chess piece source/license documentation is present.
* OpenGameArt sound source/license documentation is present.
* Lichess/FEN source documentation is present.
* Font Awesome Free, Google Fonts / Inter, Flutter package, Android Play Core, and AndroidX multidex notices are documented.
* Launcher icon branding is documented as Turbo Chess project branding.
* Private signing files and generated release outputs are excluded from the public source repository.

## Public source references

* Root license: `LICENSE`
* GPL copy: `COPYING`
* Root notices: `THIRD_PARTY_NOTICES.md`
* App legal assets: `assets/legal/`
* Stockfish metadata: `assets/stockfish/STOCKFISH_SOURCE.txt`
* Stockfish build notes: `assets/legal/STOCKFISH_BUILDING.md`

## Remaining release checks

Maintainers should re-check Stockfish GPLv3/source availability, hosted privacy policy content, Play Console forms, and app-store metadata whenever distributing public app binaries.
